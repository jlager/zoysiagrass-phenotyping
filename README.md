# Zoysiagrass Phenotyping

Tools for phenotyping *Zoysia* (zoysiagrass) experiments, with a focus on image-based trait extraction.

This repository is intended as a working space for:
- Running analysis notebooks

---

## Repository structure

- `notebooks/` – Jupyter notebooks for analysis and reports  
- `data/` – Raw data (`.gitignore`d for size)  
- `scripts/` – Reusable Python scripts for data cleaning and analysis
